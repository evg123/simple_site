Post Mortem

My platformer was a lot of fun to work on, and I was able to accomplish a lot in the allocated time, but there is still plenty of work to be done.

The enemy AI is very basic as of release 1.0. Enemies don't take into account the position of the player, or any aspects of the world geometry. I'd like to add a new algorithm that allows enemies to take into account where their actions would move them when selecting the next action they should take.

I built a flexible gui/menu system into the game, but then only used it for the status bar and pause menu. It could be used to add information displays and configuration to the game relatively quickly, and its ability to group elements of the GUI together to be shown/hidden all at once helps with organizing the code. Adding option menus and character sprite selection menus would be a nice addition in the future.
